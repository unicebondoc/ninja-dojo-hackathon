# Ninja Dojo Notion Schema

Real Notion sync is optional. The app must keep working when these databases or
the Notion token are missing.

## Projects

- Name (title)
- Status
- Type
- Priority
- Project ID
- Current Mission
- Last Receipt
- Next Action
- Created At
- Last Updated

## Missions

- Name (title)
- Status
- Mission ID
- Run ID
- Project ID
- Scroll Text
- Assigned Departments
- Active Agent
- Meowts Score
- Receipt ID
- Started At
- Completed At

## Receipts

- Name (title)
- Status
- Receipt ID
- Run ID
- Project ID
- Mission ID
- Summary
- Score
- Receipt URL
- Created At
